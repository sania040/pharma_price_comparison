# 💊 Medicine Price Checker Tool

## 🛠 How to Use

1. Install required packages:
   pip install beautifulsoup4 pandas

2. Run the script:
   python price_checker.py

3. Type a medicine name to search.
   Type 'exit' to quit.

## 📁 Folder Structure

- price_checker.py — Main script
- html_files/ — Put your WhatsApp .html files here
- 2OMAY.html — Example file included
